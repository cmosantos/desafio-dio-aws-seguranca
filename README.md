# Desafio DIO: Implementações de Segurança na AWS

Este repositório foi criado para um desafio da DIO e apresenta um relatório técnico com três implementações essenciais de segurança na AWS. O conteúdo está organizado em módulos para facilitar entendimento e aplicação prática, cobrindo controle de identidades e acessos, proteção de dados e monitoramento para detecção e resposta.

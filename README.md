# Desafio DIO: Implementações de Segurança na AWS

Este repositório foi criado para um desafio da DIO e apresenta um relatório técnico com três implementações essenciais de segurança na AWS. O material foi organizado em módulos para facilitar o entendimento e a aplicação prática, cobrindo controle de identidades e acessos, proteção de dados e mecanismos de monitoramento para detecção e resposta.

O objetivo é consolidar boas práticas e uma visão de segurança em camadas, alinhada ao modelo de responsabilidade compartilhada, fortalecendo governança e reduzindo riscos em ambientes de nuvem.
